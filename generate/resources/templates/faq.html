Why isn't there a description for X?
Descriptions are just about the one thing that can't be automated. They have to be written manually.
Can I help?
Sure! You can contribute by forking and submitting a pull request to this project's repository. See the CONTRIBUTING file for information on how to write descriptions.
Is the information provided here accurate?

It is as accurate as I can make it. When writing up or reviewing the description for a particular API item, I research the item to make sure it is described as correctly and thoroughly as possible (this is apart of why there are so few descriptions).

If you find a piece of information that is incorrect or out of date, let me know by submitting an issue.
